## Interfaz de carga de datos

Para automatizar la carga de información en Google Sheets, se desarrolló una aplicación web que permite registrar los datos de las cenas de forma estructurada, evitando el ingreso manual en hojas de cálculo.

### Funcionalidades
- Registro de asistencia por persona  
- Carga de información de la cena (comida, precio, ubicación)  
- Envío automático de datos a Google Sheets mediante API  

### Tecnologías utilizadas
- Python (Flask)  
- Google Sheets API  
- Deploy en PythonAnywhere
